# Summary

Virtuous Contract - 2B starter sword.
Model ripped from game, than resized and repaired. Use as reference to make cosplay prop.

**Trivia:**
In game, sword length is 140cm + 20cm tassels. That's corresponding to 2B height - 160 cm + 8cm heels. However, props, available on market, is only 120cm length.
